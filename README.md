-> Trabajo Práctico - Proyecto de Tareas (Spring Boot)

-> Tecnologías utilizadas
- Java 17  
- Spring Boot  
- Spring Data JPA  
- Gradle  
- H2 Database (entorno de desarrollo)

-> Estructura del proyecto
- **Model:** contiene las clases principales (`Tarea`, `Prioridad`).
- **Repository:** maneja el acceso a datos mediante JPA.
- **Service:** incluye la lógica de negocio y ejemplos de inyección de dependencias.
- **Application:** punto de entrada del sistema (`TareasApplication.java`).
